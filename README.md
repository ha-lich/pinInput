# The PIN

## Requirements

- [Node 18.x](https://nodejs.org/dist/v18.16.0/node-v18.16.0-x64.msi)

## Following the steps below to install project on development

 - Copy `.env.example` to `.env`
```shell script Linux
cp .env.example .env
```

 - Install package:
 ```shell script
yarn -i
```

 - Run project fe
 ```shell script
yarn dev
```

 - Run json server
 ```shell script
yarn json-server
```

 - Run test
 ```shell script
yarn test
```
